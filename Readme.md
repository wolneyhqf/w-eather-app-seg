# W-Eather App

Projeto prático desenvolvido para a disciplina de **Dispositivos Móveis** da Unifavip. O objetivo é aplicar conceitos trabalhados em aula por meio da construção de um aplicativo móvel com **React Native** utilizando o ecossistema **Expo**.

## Pré-requisitos

- Node.js 18+ e npm instalados
- Expo CLI (`npm install -g expo-cli`) ou uso do `npx`
- Aplicativo **Expo Go** instalado no dispositivo móvel (opcional, recomendado para testes rápidos)
- Coloque no .env sua chave de API obtida no OpenWeather (https://openweathermap.org/api)

## Passos para execução

1. Instale as dependências do projeto:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npx expo start
   ```
3. Escolha a plataforma desejada conforme comandos mostrados no terminal após iniciar o servidor de desenvolvimento

## Estrutura básica

- `src/screens`: Telas principais do aplicativo
- `src/components`: Componentes reutilizáveis
- `src/services`: Integrações com APIs e regras de negócio
